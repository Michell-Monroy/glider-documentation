Its a web interface where anyone can search some release(s)/track(s)

The content of this api is obtained from \doxylink{catalogue__workflow_8md}{Catalogue Module}

The main tools in the API are\+:
\begin{DoxyItemize}
\item /release (get)
\item /v2/release (get)
\end{DoxyItemize}

The first one executes a search of a single one release or track using an existing upc or isrc respectively. ~\newline
 The second one also makes releases/tracks searching but in massive way.

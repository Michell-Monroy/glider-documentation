Generates a csv and pdf file which contains all the processed lines of one or more specifics artist/label/date.

This feature is possible only when client has a catalogue with us.

The general performance is the following\+:


\begin{DoxyEnumerate}
\item Select just the required fields/columns from Mongo\+DB
\item Extract only columns shown previously
\item Generate csv
\item Compress csv to gzip file
\item Search client/organization info (Royalty percentage)
\item Generate PDF file
\item Sent to user

Uploads module is composed of these submodules\+:


\begin{DoxyItemize}
\item Google Drive
\item Upload File Manually
\item FTP
\end{DoxyItemize}

Each module has their own features (explained in each section) but in at general performance, all of them do the following\+:
\begin{DoxyEnumerate}
\item Receives reports
\item Decompress gzip, zip files
\item Skips non txt, csv, xls, tsv files
\item Normalizes filename (replaces blank spaces by underscores)
\item Fits csv, xls reports to csv standard
\item Uploads reports to s3 The 2-\/5 steps will called \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}"{}\+Cleaning Procedure"{}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

\item Activate env variables. You should create two .env files

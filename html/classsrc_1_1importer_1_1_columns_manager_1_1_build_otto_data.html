Detailed Description

Builds a homogenized dataframe according to ottomapping field of each format in out database 
Checks each ftype and applies different instructions depending of ftype:
ftype = header: Creates a new columns using field "name".
ftype = fixed_value: Creates a new column which contains only field "value".
ftype = custom_code: Creates a new column using the column given by format_type.
ftype = ignore: Creates a empty column using field "name".
